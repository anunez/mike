Mike - Lean Task Runner

Mike

Lean task runner

GitHub repo

An easy to use and understand task runner.

Get Started Quickly

Run npm i -g mike and create a mikefile.js file.

Fast

The commands within a task are run in parallel, speeding up the process.

Concise

Take the best from Makefile with a JS syntax.

Pre & Post tasks

Define tasks to be run right before or after a given one.

Example

module.exports = {
  'clean': {
    'cmd': ['cat .gitignore | xargs rm -rf']
  },
  'npm:install': {
    'out': ['node_modules'],
    'cmd': ['npm install']
  },
  'css': {
    'pre': ['npm:install'],
    'cmd': [
      'node_modules/.bin/lessc less/main.less --clean-css="--s0 --advanced" --autoprefix="last 2 versions"'
    ]
  },
  'js': {
    'pre': ['npm:install'],
    'cmd': [
      'mkdir -p static',
      'node_modules/.bin/browserify js/main.js | node_modules/.bin/uglifyjs --compress --mangle > static/script.js'
    ]
  },
  'build': {
    'pre': ['css', 'js']
  }
}
